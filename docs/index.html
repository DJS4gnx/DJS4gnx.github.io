Hi, welcome to my website. This is about the tenth evolution. After using Rentry for a while, I decided to get back into having my own little page.
This website is meant in mind for 1920x1080 as that's my device resolution, so mobile devices may run into issues, specifically on this page. Be warned! And please change your device screen to Landscape, if you can.

If you're looking for some stuff to read (since I write a lot), or you just want to know right now where the fun is, here's some places to check out:

  • My blog! I write long form posts here, such as retrospectives, long rants and yaps about stuff I like!
  • I have a Journal/Miniblog/Status hybrid page where I post shorter blog posts in a Tumblr-esque style, based on Nami's template. Specifically using a bunch of dumb tags. Also, I'll try and post my demos there!
  • My resources page, especially if you want to look at new software!

All site source is available on GitHub to comply with open source licenses and requests from template authors. sincere apologies for any warcrimes, i'm rusty!
