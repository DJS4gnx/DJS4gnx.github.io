Welcome to my website!

Hello! This is the tenth evolution to my personal website, we have landed here. After sticking to Rentry for a while, I decided I wanted a personal website again!
If you'd rather a rentry, I still have mine set up here!

If you come from NeoCities, hello! This was originally a GitHub pages domain, and since I didn't yet entirely prepare for that, sometimes you might get redirected to Github.io. That should be fine, I'll update these links as I find them, hopefully.
If you need to reach me, feel free to reach out through NeoCities itself, but I might not always message you that way, so it's recommended to go through SpaceHey too, if you can.

All site source is available on GitHub to comply with open source licenses and requests from template authors. sincere apologies for any warcrimes, i'm rusty!
The only two official domains for my website is djs4gnx.github.io and djsng.neocities.org.
