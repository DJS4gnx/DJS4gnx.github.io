This Website

So, this website... It has a very interesting history. In November 2020, I was doing test stuff with HTML that was entirely unstyled, basically some junk. Starting from about April 2021 through late 2022, I picked up a copy of the abandoned 2010 site MetalMIDIs and modified it to my own needs, like a fellow friend's site. For a while, it was an inside joke with a bunch of my friends that my website was a crappier, inferior competitor to his site. I'm not sure if I can provide any names, but basically, it was a harmless minibeef.

When I came around to Neocities around 2023, I picked back up by using some free site templates. Goblin Heart, Nami's templates... All that. (WIP)

And after picking up a Kalechips layout, we're here. Woo, back on the grind!
